Object Types

General object types is same as Apollo Server. Each field inside of an object type maps to another type, allowing nested types and circular references.

type TypeName {
  fieldA: String
  fieldB: Boolean
  fieldC: Int
  fieldD: CustomType
}

type CustomType {
  circular: TypeName
}
