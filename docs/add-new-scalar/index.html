Add new Scalar

You can define new scalar like Apollo Server.

  1. Define scalar name in SDL.
scalar MyCustomScalar
  1. Define how to serialize MyCustomScalar. You can see more information in GraphQLScalarType.
// myCustomScalar.js
const { GraphQLScalarType } = require('graphql')

module.exports = new GraphQLScalarType({
  name: 'MyCustomScalar',
  description: 'my custom scalar',
  serialize: ...,
  parseValue: ...,
  parseLiteral: ...,
})
  1. Map new scalar in GQLify.
const { Gqlify } = require('@gqlify/server')
const MyCustomScalar = require('./myCustomScalar')

const server = new Gqlify({
  sdl: ...,
  dataSources: ...,
  scalars: {
    MyCustomScalar
  }
})
